# DSU_BoatControl
 이 프로젝트는 친구의 졸업 작품을 돕기 위해 만들어졌습니다. 
 프로젝트의 주요 내용은 객체 인식 프로그램을 활용하여 자율적으로 움직이는 쓰레기 수거 보트를 개발하는 것입니다. 
 저는 프로젝트에서 객체 인식 프로그램과 보트의 조종 시스템을 코딩을 지원해줬습니다.

 우리는 YOLOv5를 객체 인식 프로그램으로 선택했습니다.
 > 선택 이유: 라즈베리 파이를 실행 플랫폼으로 사용하기로 했기 때문에 YOLOv5를 선택했습니다.

 친구의 작품을 도와주는 이유 : 라즈베리파이 사용 공부 , 객체 인식 프로그램 공부
* * *

 This project was created to assist my friend with their graduation project. 
 The main focus of the project is to develop an autonomous garbage collection boat using an object detection program. 
 I supported the project by coding the object detection program and the boat's  control system.

 We chose YOLOv5 as the object detection program.
 > Reason for the choice: We decided to use Raspberry Pi as the execution platform, so we chose YOLOv5.

 Reason for helping my friend with their project: To study Raspberry Pi usage and object detection programming.
